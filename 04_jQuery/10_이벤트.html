이벤트 달자~~

* 방법 1. 이벤트 메소드 호출해서 연결

        $("선택자").이벤트메소드(function(){
            이벤트 발생 시 실행할 코드;
        });

클릭해보세요 ~

* 방법 2. on메소드를 호출하는 방법

        [표현식]
        $("선택자").on('이벤트타입', function(){
            이벤트 발생시 실행할 코드;
        });

마우스로 클릭도 해보고~ 올려도 보고~

* 방법 3. ☆★on 메소드를 사용하는 방법 ☆★

        [표현법]
        $("상위요소선택자").on("이벤트타입", "하위요소선택자", function(){
            이벤트 발생 시 실행할 내용;
        });

h4요소를 클릭해보세요.

* 키보드 관련 이벤트

        키가 눌렸을 때 발생하는 이벤트
        - keydown : 키보드의 모든 키가 눌리면 발생
        - keypress : 키보드의 function키, 기능키, 한글을 제외한
                     나머지가 눌릴 때 발생 



        키가 떼어질 때 발생하는 이벤트
        - keyup
