Content영역과 관련된 메소드

        선택된 요소의 content영역(여는태그와 닫는태그 사이)를 반환해주거나
        또는 변경해주는 메소드들 


        - $("선택자").html()
        - $("선택자").text()
        - 선택된 요소의 content영역 값을 반환해줌 

        - $("선택자").html("인자값")
        - $("선택자").text("인자값")
        - 선택된 요소의 content영역의 값을 전달한 인자값으로 바꿔줌 


        => 이때, html()는 HTML태그를 인식

* 추가 관련 메소드

        선택된 요소를 기준으로 새롭게 생성된 요소를 추가해주는 메소드

        $(A).append(B) : A요소 내부에 가장 뒷부분에 B를 추가(자식) 

        $(A).prepend(B) : A요소 내부에 가장 앞부분에 B를 추가(자식) 

        $(A).before(B) : A요소 앞 부분에 B를 추가(동위) 

        $(A).after(B) : A요소 뒷 부분에 B를 추가(동위) 

        
        


        $(B).appendTo(A) : B를 A요소 내부에 뒷부분에 추가(자식) 

        $(B).prependTO(A) : B를 요소 내부에 앞부분에 추가(자식) 

        $(B).insertBefore(A) : B를 A요소 앞부분에 추가(동위) 

        $(B).insertAfter(A) : B를 A요소 뒷부분에 추가(동위)

* 요소객체 제거 / 잘라내기 메소드

$("선택자").empty() : 선택된 요소의 모든 후손요소 제거
$("선택자").remove() / detach() : 해당 요소 잘라내기
- remove : 요소의 이벤트가 날아가버림
- detach : 이벤트가 같이 잘라내짐
